# Serenity Mirror 🪞

> 一个帮你「盯住一个高手、读懂他、甚至预判他」的自动追踪系统。
>
> 围绕半导体 / AI 供应链投资分析师 **Serenity（@aleabitoreddit）** 打造：他在 X（推特）一发帖，你的飞书立刻收到**中文翻译 + AI 点评 + 买卖信号**；系统还会把他每个观点存档、到期**联网核对他当时说得对不对**，并据此不断进化出一个越来越懂他的「思维分身」。
>
> ⚠️ 仅供研究学习，所有内容均为 AI 推断，**不构成任何投资建议**。

## 这个项目帮你解决什么

盯一个高产、全英文、信息量巨大的博主，人工根本跟不过来。这个系统帮你做四件事：

1. **不漏帖**：他一发新帖，几分钟内自动推到你飞书，再也不用手动刷。
2. **看得懂**：英文帖自动翻成地道中文，还附一句 AI 点评，告诉你这条在说什么、重不重要。
3. **抓重点**：自动判断每条帖是「🔴 强烈看多 / 🟠 值得关注 / ⚪ 普通观点」，只有重要的才弹给你，普通的进存档，不打扰。
4. **读懂他**：把他的每个观点记下来，过段时间联网查真实结果（股价、财报、事件），看他当时说得对不对，慢慢摸清他「哪类判断靠谱、哪类爱翻车」。

## 你会收到什么

**① 实时帖子卡片（飞书）**
每条重要新帖一张卡片：左上角圆形头像 + 博主中文名和简介 → 中文翻译全文 → AI 点评（含买卖信号，强信号标红）→ 一键跳转原帖。带图带视频封面。

**② 每日两次日报（飞书，早 8 点 / 晚 8 点）**
把过去半天的帖子汇总成一份日报，按市场分区（美股 / A股 / 港股 / 韩股）列出「值得买入 / 值得关注」的股票和方向，并点出多位博主都在聊的「共识方向」。

**③ 可筛选的存档表格（飞书多维表格）**
每条帖都存一行，字段齐全：时间、博主、信号、市场、标的、中文翻译、AI 点评、原帖链接。你可以随时按「只看强烈看多」「只看 A股」「只看今天」筛选回看。

**④ 每周「懂他复盘」（飞书）**
每周一份复盘：他近期哪些判断应验、哪些翻车，一个「懂他指数」（我们预判他的准确率），以及他下一步可能关注的方向。

## 它最特别的地方：会「读懂」也会「自我进化」

普通的盯帖工具只是搬运。这个系统多了一个**学习闭环**，这是它的灵魂：

- **记观点**：他每发一个有具体标的的观点，系统提炼成一张「主张卡」，还会补全他没明说的「为什么这么看」。
- **查对错**：到了约定时间（2 周 / 1 个月 / 1 季度），系统**自动联网查真实结果**——股价涨跌、有没有拿到订单、财报兑现没有——判定他当时说得「应验 / 部分应验 / 证伪」。
- **纠偏**：把对错累积成一张「可信度地图」，知道他哪类逻辑更可信、哪类要警惕。
- **越来越懂他**：用真实结果给我们「预判他」打分，分数越攒越准，系统也据此不断修正那个「思维分身」。

## 附带产物：`serenity-master`（一个可直接用的「思维分身」）

仓库里还打包了一个 **`serenity-master`** —— 一个可以直接喂给 AI 助手（Codex / Claude 等）的「Serenity 思维操作系统」。装上之后，你可以让 AI「用 Serenity 的视角」帮你分析任何一只股票，或者直接模仿他本人的口吻说话。它由以下几块组成：

### 1. 5 个核心心智模型（他怎么想）
他做判断的底层逻辑，每条都附「证据 + 怎么用 + 局限/反例」，不只吹优点：
- **供应链卡点**：最赚钱的机会在那些控制「别人离不开的关键零件」的小公司。
- **瓶颈估值**：这类公司不能按营收估值，要问「它断供会怎样」。
- **NVIDIA 信号**：英伟达投什么方向，那个方向的关键环节迟早兑现。
- **信息不对称**：机会在「机构嫌太小、散户看不懂」的夹缝里——这是他真正的本事。
- **正和市场**：不碰期权，长期拿住好公司是正和游戏。

### 2. 8 条决策启发式（他怎么做决定）
可直接照用的决策规则，连他自己的「言行矛盾」也如实标出（比如嘴上反期权、行动却用期权打法）：卡点测试、跟随英伟达、偏好欧洲小盘、反「妖股」标签、机构跟进确认、反期权、DYOR 底线、地缘政治溢价。

### 3. 方法论引擎（他怎么做研究）
一条完整研究路径：从市场叙事 → 找到最稀缺的环节 → 锁定相关公司 → 分级证据 → 想清楚「什么情况说明判断错了」→ 按证据强弱决定下不下注。附带打分表和估值方法。

### 4. 表达 DNA（他怎么说话）
扮演他时的语言风格：结论先行、用数据说话、对杠精直接怼、不用「也许大概」这种含糊词。

### 5. 四档可信度（防 AI 瞎编）
AI 说的每句话都标清来源：① 他的原话 ② 归纳出的规律 ③ 框架推断（必须说明「这是推断不是结论」）④ 没依据就直说，绝不硬编。

### 6. 自带质量自检
内置三类测试题（已知案例 / 全新领域 / 风格还原），每次更新后给自己打分，保证不跑偏。

### 7. 两种用法
- **顾问模式（默认）**：第三人称帮你拆解「用他的框架看，他会怎么想」。
- **扮演模式**：直接用「我」以他的身份回答。
- 遇到「梭哈 / 借钱加杠杆」这种高风险问题，自动退出扮演、给风险提示。

### 8. 自我进化（evolution/）
这个思维分身不是死的：它由上面的「读懂闭环」持续喂养。核心人设（心智模型、说话风格）的改动需要人工确认；最新持仓、观点、可信度地图这些会自动刷新、留痕、可回退，避免越学越歪。

## 想自己跑起来？

这套系统需要一点技术基础，简单说三步（细节问作者）：

1. **准备一台能上推特的境外服务器**，装好运行环境。
2. **填一份配置**：把 `config.example.json` 复制成 `config.json`，填入你自己的 X 登录信息、飞书机器人、AI 翻译接口、飞书表格地址（都是你自己的，不会泄露给别人）。
3. **设定时任务**：让它按时自动跑（盯帖每 5 分钟、日报早晚各一次、验证每天一次、复盘每周一次）。

> 股价数据：A股用 [a-stock-data](https://github.com/simonlin1212/a-stock-data)、港美股用 [global-stock-data](https://github.com/simonlin1212/global-stock-data)，均已集成，免费无需鉴权。
